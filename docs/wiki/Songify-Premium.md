# Songify Premium

Songify Premium adds **optional** features and helps fund development. The core app stays **free**; Premium does not remove free functionality.

---

## Activate Premium

1. Become a [Ko-fi](https://ko-fi.com) supporter (any tier).
2. Visit [songify.rocks](https://songify.rocks) and log in.
3. Open your account page → **Link Ko-Fi**.
4. Use the **same email** as on Ko-fi.
5. Generate your **Songify token** and add it in the app.

**Settings → System → Songify Token**, or use **Import to Songify** on the website when offered.

---

## Cloud sync

Sync settings across PCs: upload, compare, and restore configuration from the cloud (requires Premium as implemented on the site).

---

## Widgets

Premium widget styles are listed on [songify.rocks/widgets](https://songify.rocks/widgets/).

Free widgets remain available, including:

- Basic now playing widgets
- Basic queue display
- [Widget Generator](https://widget.songify.rocks)

---

## Token security

- The token is shown **once** when generated.
- If lost, **rotate** and generate a new token on the website.
- The token authenticates Premium features and cloud sync.

---

## Privacy

Ko-fi membership is validated via webhooks. Emails used for matching are handled in line with the privacy policy on the Songify site.
